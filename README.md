# 🌱 Fertilizer Recommendation System

A Machine Learning based web application that recommends the most suitable fertilizer for crops based on soil nutrients, crop type, and environmental conditions.

This system helps farmers make data-driven decisions to improve crop productivity and maintain soil health.

---

## 🚀 Features

- Predicts the most suitable fertilizer for crops
- User-friendly web interface
- Machine learning model integration
- Soil nutrient analysis (Nitrogen, Phosphorus, Potassium)
- Fast prediction using trained model

---

## 🛠️ Technologies Used

- Python
- Flask
- Scikit-learn
- Pandas
- NumPy
- HTML
- CSS
- Jupyter Notebook
